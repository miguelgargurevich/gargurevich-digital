## Tips for best results

- **Aspect ratio**: 16:9 or wider (landscape)
- **Recommended size**: 1280×720 px minimum
- **Format**: PNG or WebP (JPG also works — just rename the `image` field in `PortfolioSection.tsx`)
- **Crop**: Use the top of the page / dashboard as the focal point (`object-top` is applied)

When a file is present it automatically replaces the placeholder mock UI shown in the card.
